# cake-marginalia

[Cake](https://github.com/flatland/cake) plugin for [Marginalia](https://github.com/fogus/marginalia).

## Usage

* Add `[cake-marginalia "0.6.0]` to your project.clj's :dev-dependencies.
* `$ cake marg` in your project's root.
